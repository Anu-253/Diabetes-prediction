# 🩺 Diabetes Risk Prediction

A machine learning project that predicts the likelihood of diabetes using medical diagnostic data from the Pima Indians Diabetes Dataset.

## 🚀 Overview

This project focuses on building and evaluating classification models for diabetes prediction using Python and scikit-learn. The workflow includes:

* Data preprocessing
* Handling missing values
* Exploratory analysis
* Model training and evaluation
* Performance comparison

The goal was to understand how machine learning can assist in early risk prediction using healthcare datasets.

---

## 📌 Features

* Cleaned and preprocessed real-world medical data
* Handled zero-valued missing features using median imputation
* Trained multiple ML models for comparison
* Evaluated model accuracy using cross-validation
* Visualized feature importance and prediction results

---

## 🛠️ Tech Stack

* Python
* pandas
* NumPy
* scikit-learn
* Matplotlib
* Jupyter Notebook

---

## 📊 Dataset

* Pima Indians Diabetes Dataset
* 768 patient records
* Features include:

  * Glucose
  * BMI
  * Blood Pressure
  * Insulin
  * Age

---

## 🤖 Models Used

* Logistic Regression
* Random Forest Classifier

Random Forest performed better during evaluation and achieved around **79% accuracy**.
